Fonctionnalités

Imports passés

Sur cette page vous verrez les imports passés et vous serez capable de les sauvegarder afin d’en créer un modèle de correspondances.

Le modèle sauvegardé pourra être réutilisé lorsque vous configurerez un nouvel import, peu importe le type de ressources.

Modèle de correspondances

Depuis cette page vous pouvez voir vos modèles sauvegardés, les supprimer ou changer leur nom.
